# 🚨 Real-Time Disaster Response Coordinator

## 📌 Project Overview

The **Real-Time Disaster Response Coordinator** is a web-based emergency management system designed to connect victims and volunteers instantly during disasters.

This system enables real-time communication, location tracking, and smart volunteer assignment to improve response time and save lives.

---

## ❗ Problem Statement

During natural disasters or emergencies, victims often struggle to receive timely help due to lack of coordination between responders and volunteers. Traditional systems lack real-time visibility of emergency requests and available resources.

---

## 💡 Solution

This platform provides a **real-time disaster response system** that:

- Allows victims to send emergency requests
- Registers volunteers with live location
- Automatically assigns nearest volunteers
- Displays requests on live dashboard
- Helps administrators monitor all activities
- Provides analytics for decision-making

---

## 🚀 Features

- 🆘 Emergency Request Submission
- 🤝 Volunteer Registration System
